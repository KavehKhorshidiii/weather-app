###### Installation Steps 

1. Clone the project 
```bash
git clone https://github.com/KavehKhorshidiii/weather-app.git
```
2. Move into the project directory
```bash
cd my-app
```
3. Install dependencies
```bash
npm install
```
4. Run Tailwind in watch mode
```bash

```
5. Start the development server
npm start

npm start
