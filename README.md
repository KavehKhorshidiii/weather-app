## 📦 Installation & Usage

###### Requirements 
- Node.js 
- npm or yarn

###### Installation Steps 

1. Clone the project 
```bash
git clone https://github.com/Kaveh-Khorshidi/AdminDashboard.git
```

3. Move into the project directory
```bash
cd my-app
```

5. Install dependencies
```bash
npm install
```

7. Start the development server
npm start
```bash
npm start
```
